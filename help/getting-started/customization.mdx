## Theme

You can select what the theme of your booking pages would be. You can either force the booking page to be of Light mode or Dark mode where the visitors to your booking page would only ever see the selected mode. Alternatively, you could select the System default (selected by default by Cal.com) where the visitors to your website would see the version as per their system default settings (Light or Dark)

## Brand Colors

You can select your Brand Color for both light theme and dark theme which over-rides the default brand color of Cal.com in your public booking pages. This adds a custom feel to the booking experience.

[![](https://downloads.intercomcdn.com/i/o/1198206313/f30357693da318f7934b7c88/i817x190--uQ-QNjQv7N-_dhazio.png?expires=1734714000\&signature=e14a0ad67e98e7ab8c19d36d1c27d463b9f42d6b7634aec5956584e468678801\&req=dSEuHst%2Bm4JeWvMW1HO4zSGYuaZeLpetOwWwqy61UKiW0pTsmTcvhb3LU8W2%0As5OsVpLHL2I9NeTC5nk%3D%0A)](https://downloads.intercomcdn.com/i/o/1198206313/f30357693da318f7934b7c88/i817x190--uQ-QNjQv7N-_dhazio.png?expires=1734714000\&signature=e14a0ad67e98e7ab8c19d36d1c27d463b9f42d6b7634aec5956584e468678801\&req=dSEuHst%2Bm4JeWvMW1HO4zSGYuaZeLpetOwWwqy61UKiW0pTsmTcvhb3LU8W2%0As5OsVpLHL2I9NeTC5nk%3D%0A)

You can easily use these in combination with [embed](https://cal.com/docs/core-features/embed) to make the cal.com embed feel like a part of your website by simply updating these settings to match your website.
