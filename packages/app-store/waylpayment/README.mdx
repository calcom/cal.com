## Wayl Payment Gateway

[Wayl](https://wayl.io) is an Iraqi payment gateway that allows merchants to accept payments via ZainCash, FastPay, and other local Iraqi payment channels.

### Features

- Accept payments in Iraqi Dinar (IQD)
- "Pay to Book" — customers pay before their booking is confirmed
- Supports ZainCash, FastPay, and other Iraqi payment channels
- Automatic booking confirmation after payment
- Automatic refunds when bookings are cancelled

### How to set up

1. Create a merchant account at [wayl.io](https://wayl.io)
2. Get your API key from your [Wayl dashboard](https://dashboard.wayl.io/settings/api)
3. Install the Wayl app from the Cal.com App Store
4. Paste your API key and click **Connect**
5. In any event type, go to **Payment** → enable **Require Payment** → set your price in IQD

### How it works

When a customer books an event that requires payment:
1. They fill in the booking form and click **Confirm**
2. They are redirected to Wayl's secure payment page
3. They pay using their preferred Iraqi payment method
4. Wayl notifies Cal.com automatically
5. The booking is confirmed and both parties receive confirmation emails

### Webhook configuration

No manual webhook configuration is required. When a customer books a paid event, the
integration automatically:

1. Generates a unique cryptographic secret for that payment
2. Creates the Wayl payment link with that secret embedded
3. Wayl signs its webhook request with the same secret
4. Cal.com verifies the signature on receipt before confirming any booking

The only environment variable required is:

```
NEXT_PUBLIC_WEBAPP_URL=https://your-cal-domain.com
```

This is used to build the absolute webhook callback URL sent to Wayl.
