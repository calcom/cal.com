Quickly share your [Cal ID](https://cal.id) meeting links with Raycast. Requires [Raycast.com](https://www.raycast.com/) to be installed and the [Cal ID](https://cal.id) from the Raycast Store needs to be installed: https://www.raycast.com/cal/. You can create an API token in your Developer [Cal ID](https://cal.id) Settings, and then you can add the API Key by opening up the [Cal ID](https://cal.id) App in Raycast.
