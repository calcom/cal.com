# BigBlueButton

BigBlueButton is a free, open-source web conferencing system designed for online learning and team collaboration. It provides high-quality audio and video conferencing, screen sharing, shared whiteboards, chat, breakout rooms, and meeting recordings — all hosted on your own infrastructure.

## Key Features

- 🎥 **HD video conferencing** — multi-participant video calls
- 🎙️ **Crystal-clear audio** — WebRTC-based, no plugins required
- 📺 **Screen sharing** — share your entire screen or specific applications
- 📝 **Interactive whiteboard** — collaborate in real-time with shared drawing tools
- 💬 **Public & private chat** — integrated meeting chat
- 🗂️ **Breakout rooms** — divide participants into smaller groups
- ⏺️ **Meeting recordings** — automatically record and play back sessions
- 🔐 **Self-hosted** — run on your own servers for complete data sovereignty

## Setup

You'll need a BigBlueButton server (v2.5+). Provide the server's API base URL and shared secret to get started.
