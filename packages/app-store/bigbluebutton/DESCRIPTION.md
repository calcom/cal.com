# BigBlueButton

BigBlueButton is an open source web conferencing system for online learning.

## Features

- **Real-time audio and video** - High-quality WebRTC audio and video
- **Screen sharing** - Share your screen with participants
- **Presentation tools** - Upload and annotate presentations
- **Breakout rooms** - Split participants into smaller groups
- **Polling** - Engage participants with live polls
- **Chat** - Public and private chat messaging
- **Recording** - Record sessions for later playback

## Setup

1. Install the BigBlueButton app from the Cal.com app store
2. Configure your BigBlueButton server URL and shared secret
3. Select BigBlueButton as your meeting location when creating events

## Configuration

You need the following information from your BigBlueButton server:

- **Server URL**: Your BigBlueButton server URL (e.g., https://bbb.example.com)
- **Shared Secret**: The shared secret from your BBB server's `bbb-conf --secret` command
