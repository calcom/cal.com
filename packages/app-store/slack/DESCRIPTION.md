Send Cal.com booking notifications to Slack. Connect your Slack workspace with a Bot Token and optional Channel ID, then add a webhook for the Slack app to receive booking created, cancelled, and rescheduled events in your chosen channel.

**Get started:** Install the app, then open Setup to add your Slack Bot Token and Channel ID. Configure webhooks in Developer settings to send events to Slack.
