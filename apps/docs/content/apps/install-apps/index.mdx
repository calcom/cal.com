# Install Apps

Install desired apps in your self-hosted instance for better integration.

- [Google](/apps/install-apps/google) - Install Google for supporting Google Calendar and Google Meet integrations
- [Microsoft](/apps/install-apps/microsoft) - Install Microsoft to support Exchange365 integration
- [Zoom](/apps/install-apps/zoom) - Install Zoom to support Zoom video conferencing integration
- [Daily](/apps/install-apps/daily) - Install Daily to support Cal Video integration
- [HubSpot](/apps/install-apps/hubspot) - Install HubSpot to support HubSpot CRM integration
- [SendGrid](/apps/install-apps/sendgrid) - Install SendGrid to support SendGrid Email integration
- [Stripe](/apps/install-apps/stripe) - Install Stripe to support Stripe integration for paid bookings
- [Twilio](/apps/install-apps/twilio) - Install Twilio to support Workflows integration for sms reminders
- [Zoho](/apps/install-apps/zoho) - Install Zoho to support Zoho CRM integration
