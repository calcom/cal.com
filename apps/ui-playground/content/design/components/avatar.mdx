## Usage Example

Here's a complete example showing how to use the Avatar component with all its props:

```tsx
import { Avatar } from "@calcom/ui/components/avatar";

function Example() {
  return (
    <div className="flex flex-col gap-4">
      {/* Basic Avatar */}
      <Avatar
        alt="User Name"
        title="User Name"
        size="md"
        shape="circle"
        imageSrc="https://example.com/avatar.jpg"
      />

      {/* Avatar with Link */}
      <Avatar
        alt="User Name"
        title="User Name"
        size="lg"
        shape="square"
        imageSrc="https://example.com/avatar.jpg"
        href="/profile"
      />

      {/* Avatar with Tooltip and Indicator */}
      <Avatar
        alt="User Name"
        title="User Name"
        size="md"
        shape="circle"
        imageSrc="https://example.com/avatar.jpg"
        indicator={<div className="absolute bottom-0 right-0 h-3 w-3 rounded-full bg-green-500" />}
      />

      {/* Avatar with Fallback */}
      <Avatar
        alt="User Name"
        title="User Name"
        size="md"
        shape="circle"
        fallback={<span>UN</span>}
      />
    </div>
  );
}
```
