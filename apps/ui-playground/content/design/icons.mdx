# Icon System

Our icon system is built using [Lucide](https://lucide.dev/) icons, providing a consistent and scalable set of icons across the application. All icons are available through the `Icon` component from `@calcom/ui`.

## Usage

```tsx
import { Icon } from "@calcom/ui";

// Basic usage
<Icon name="calendar" className="h-4 w-4" />

// With different size
<Icon name="user" className="h-6 w-6" />

// With different color
<Icon name="star" className="h-4 w-4 text-yellow-500" />
```
