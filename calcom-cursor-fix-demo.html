📋 About This Fix

PR: #28646

Issue: In the embed view, enabled date buttons didn't show a pointer cursor on hover, making it unclear to users that dates were clickable.

Solution: Added cursor: "pointer" to the inline styles for enabled date buttons in the embed context.

Impact: Improved user experience with clear visual feedback indicating interactivity.
